Or from the proper icon

gpapExtract Interface¶

Parts of Interface¶

Essential parts of the Plugin:

  1. Select Database
  2. Select Folder to Save
  3. Select Feature fron Note
  4. If selected, coordinates will be added in the desired crs, otherwise only the geographic coordinates will be saved
  5. If 4 is selected the crs can be obtained from Layer
  6. If 4 is selected the crs can be obtained from Project
  7. If 4 is selected the crs can be obtained from CRS Selector
  8. Once the crs is selected the code will appear here
  9. And the QGIS CRS ID will appear here
  10. The PROJ string will appear here
  11. once you have all the fields it is time to process, an the results will appear in 13
  12. Click here to clear all fields
  13. Result and file path will appear here
  14. Help (this)
  15. Cancel/Ok to exit plugin
